# Hourly Sales Analysis

## Overview
This project analyzes sales distribution across different hours of the day using Python.

## Tools Used
- Python
- pandas
- matplotlib

## Analysis
- Extracted hour from transaction timestamps
- Aggregated sales per hour
- Identified peak and low sales hours
- Visualized hourly sales patterns

## Key Insight
Sales activity is not uniformly distributed throughout the day and shows clear peak periods, indicating time-dependent customer behavior.

## Files
- `Analysis.ipynb`: Jupyter notebook containing the analysis
- `Order_details.csv`: Dataset used for analysis
